# FedEx Dry Ice for Magento 2 — by Towering Media

Ship frozen or pharmaceutical products with FedEx-compliant dry ice automation. Transmit UN1845 classifications, weight, and package counts via the FedEx REST API. Assign dry ice quantities per package to stay within carrier limits. Prompt shippers for required consignee and packaging declarations. Produce labels and documents that satisfy FedEx cold-chain regulations. Fully compatible with Magento 2.4.x Open Source and Adobe Commerce.
